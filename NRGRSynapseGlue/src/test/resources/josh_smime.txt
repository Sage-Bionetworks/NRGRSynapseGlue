Hi Nicole,

=20

As discussed, the most common form of identity proofing emails at the =
NIH is to either digitally encrypt or digitally sign emails using the =
digital signatures on the NIH PIV card. While it is possible to send =
digitally signed emails to anyone, regardless if the recipient have =
digital ID=E2=80=99s themselves, whether your digital signature can be =
verified depends of the configuration/compatibility of the =
recipients=E2=80=99 email client. The use of digital certificates and a =
password or PIN is known as, =E2=80=9Cdual-factor =
authentication=E2=80=9D as it requires having both the certificates and =
the password to use them.

=20

For example, Gmail does not have the ability to verify digital =
ID=E2=80=99s, however the digital signature will be seen as an =
attachment, usually named, =E2=80=9Csmime.p7s.=E2=80=9D While it may be =
possible to use third-party software to verify the attachment, I am =
unsure what options would be available to do so.

=20

The next option would be to use the NIH Secure Email and File Transfer =
system, which is also known as SEFT, which can be accessed from =
https://secureemail.nih.gov/bds/Login.do

=20

SEFT allows for sending digitally encrypted and secured emails to =
external collaborators without the use of PIV card =
encryption/certificates. The caveat of using SEFT is that all =
emails/attachments have a maximum set life-span of 90 days. After which, =
the information is permanently deleted. For this reason, SEFT should =
never be used to archive communications/files for any period longer than =
90 days. It should be mentioned that unlike how certificates use =
=E2=80=9Cdual factor=E2=80=9D authentication, SEFT only uses =
=E2=80=9Csingle-factor=E2=80=9D authentication where only a username and =
password are needed to access it.=20

=20

When an email is sent in SEFT, the recipient will get a notification in =
their email indicating they have a new SEFT message and a link to access =
it. If it is the first time the recipient will use SEFT (and if they are =
not within the NIH), it will have them first register a SEFT account, =
which they can then use to access and reply to the emails.=20

=20

Let me know if this information is helpful and if you have any further =
questions, comments, or concerns.=20
